# 《医学图像重建》学习路线

这份路线是按曾更生《医学图像重建》的目录结构整理的，目标是让初学者尽快建立主线、看懂图、逐步补齐公式推导，而不是从第一页开始平均用力。

参考目录来源：
- https://www.shukui.net/book/269986.html
- https://academic.hep.com.cn/foms/CN/book/978-7-04-028828-5-00

## 一、先明确这本书的主线

整本书其实围绕一条链条展开：

**物体/组织分布 f(x,y,z) -> 投影数据/频域数据怎么获得 -> 怎样反演回图像 -> 为什么会有噪声、伪影、欠采样误差 -> 解析重建和迭代重建各自怎么处理**

你读每一章时都先问一句：

**这一章是在讲“数据怎么来”、“怎么反演回去”，还是“反演为什么会坏、怎么补救”？**

如果这个定位没抓住，图和公式会很容易碎掉。

## 二、推荐学习节奏

不建议一上来就追求把所有证明一次看懂。建议分三遍：

### 第一遍：2-3周，先建立框架

目标：
- 知道每章在整条重建链条里的位置
- 知道每种算法的输入、输出、适用几何、主要缺点
- 先能“讲清楚”，不要求每一步推导都完整复现

做法：
- 以第1、2、3、4、6、7章为主，第5章先只建立概念印象
- 每节读完写3句话：
  - 这节要解决什么问题
  - 核心操作是什么
  - 为什么会产生误差/伪影

### 第二遍：3-5周，专门补公式推导

目标：
- 把高频核心推导补起来，尤其是中心切片定理、FBP、ART/MLEM/MAP
- 能区分“物理建模公式”和“数值求解公式”

做法：
- 每个推导都按这四步拆：
  1. 先写清楚已知量和未知量分别是什么
  2. 标出这一行到下一行用了什么数学工具，比如傅里叶变换、卷积定理、变量代换、梯度下降、极大似然
  3. 如果中间跳步，就自己补一行“中间桥”
  4. 最后用一句话解释这个公式在算法中负责哪一步

### 第三遍：持续进行，专门训练看图和做题

目标：
- 把书里的图从“看不懂”变成“能翻译成变量和算法步骤”
- 用例题和练习题检查自己是不是真的懂了

做法：
- 每看到一张图，固定回答4个问题：
  - 横轴、纵轴、颜色、箭头分别表示什么量？
  - 这张图是在图像域、投影域、还是频域/k空间？
  - 它是在说明成像几何、算法流程、还是噪声/伪影现象？
  - 这张图想让你记住的一句话结论是什么？
- 对看不懂的图，不要只盯图本身，要立刻回到图前后那一段文字和公式，找它对应的是哪个变量、哪一步操作

## 三、按章节的具体学习路线

## 第1章 断层成像的基本原理

### 这章定位

这是全书地基，先建立“投影”和“反投影”这两个核心概念。

### 优先吃透
- 1.1 断层成像：为什么不是直接拍一张图，而是要从多角度测量后重建
- 1.2 投影：投影数据到底是什么
- 1.3 图像重建：重建问题本质上是反问题
- 1.4 反投影：为什么单纯反投影会模糊
- 1.5.1 投影、1.5.2 反投影：先看懂变量和几何意义

### 第一遍可以先放慢但不死磕
- 1.5.3 狄拉克 δ 分布函数

这一节不要急着把分布理论完全学完，第一遍只要知道：

**δ 函数常被用来把“沿某条直线/曲线取积分”写成一个统一的积分表达式。**

### 这一章最该形成的直觉
- 投影不是图像本身，而是很多条射线路径上的累积测量
- 反投影把每个投影值“涂回去”，但如果不滤波会产生拖影/模糊
- 所以后面第2章的核心任务就是：怎么在反投影前后加正确的频域补偿

### 过关标准
你能不用书讲清楚：
**为什么“投影 -> 直接反投影”不够，为什么必须引入滤波/反演公式？**

## 第2章 平行光束图像重建

### 这章定位

这是二维CT解析重建的核心章，建议作为第一轮重点。

### 优先吃透
- 2.1 傅里叶变换：重点不是背定义，而是知道“空域卷积 <-> 频域相乘”
- 2.2 中心切片定理：这是整章最核心的一句话
- 2.3 重建算法：重点理解“直接傅里叶法、FBP、先卷积后反投影、先反投影后滤波”这些方法为什么本质上等价或近似等价
- 2.4 计算机模拟例子：专门拿来训练看图
- 2.5 利用截断的投影数据重建ROI：第一遍先知道“局部数据会导致全局不适定，但在某些条件下ROI仍可重建”

### 第二遍重点补的推导
- 2.6.3 中心切片定理的证明
- 2.6.4 FBP算法的推导
- 2.6.5 先卷积后反投影算法的数学表达式
- 2.6.7 先反投影后滤波算法的推导

### 第一遍可以先略读
- 2.6.2 希尔伯特变换与有限希尔伯特变换
- 2.6.6 Radon反演公式的数学表达式

不是说不重要，而是对刚入门的人，先把“中心切片定理 -> 滤波反投影”主线走通更划算。

### 这一章看图技巧
凡是看到频域里的线、圆、切片图，先判断：
- 这条线是不是某个角度投影的一维傅里叶变换
- 它在二维频域里对应的是不是穿过原点的一条径向切片
- 角度 θ 变了以后，是不是在频域里扫过不同方向

### 过关标准
你能讲清楚：
**中心切片定理为什么把“投影数据”和“图像二维傅里叶频谱”联系起来，以及FBP为什么要先滤波再反投影。**

## 第3章 扇形束图像重建

### 这章定位

从“平行束理想几何”走向更接近实际X射线CT的“扇形束几何”。

### 优先吃透
- 3.1 扇形束成像几何：源点、探测器、射线角度、旋转角这些变量各指什么
- 3.2 平行光束算法到扇形束算法的转换：理解“重排/加权/变量替换”这件事
- 3.3 短扫描：为什么不一定要满360度，为什么要加权

### 第二遍再补
- 3.4.1 扇形束FBP算法的推导
- 3.4.2 扇形束的DHB算法

### 这一章最容易卡的地方
你可能不是公式看不懂，而是几何角度符号混了。建议自己单独画一张“扇形束变量对照图”，把：
- 射线源位置
- 探测器坐标
- 旋转角
- 扇角
- 重建点坐标

全部标出来。很多推导其实是在做几何变量替换。

### 过关标准
你能说清楚：
**扇形束和平行束的主要区别是什么，为什么平行束FBP不能原封不动直接套上去。**

## 第4章 透射型投影和发射型投影的断层成像

### 这章定位

把前面比较抽象的重建公式，落回到具体模态：X光CT、PET、SPECT。

### 优先吃透
- 4.1 X光CT：测到的是什么，为什么常和线积分、指数衰减、对数变换联系起来
- 4.2 PET和SPECT：发射型成像和透射型CT在数据来源上有什么本质区别
- 4.3 发射型断层成像的衰减修正：为什么“组织自身对射线的吸收”会让重建偏掉

### 第一遍对 4.4 数学表达式 的建议
不要一上来陷进每个公式细节，先把每个模态对应的“测量物理 -> 重建数学模型”对上号。

### 过关标准
你能区分：
**CT / PET / SPECT 分别是在测什么量，为什么它们虽然都能做断层重建，但正向模型不完全一样。**

## 第5章 三维图像重建

### 这章定位

把二维重建推广到三维。对初学者来说，这是全书最容易感觉“图和几何突然变复杂”的一章。

### 建议学习策略
第一遍不要把这一章当主战场，先抓概念地图，第二遍再挑重点公式深入。

### 第一遍优先建立的概念
- 5.1 平行的线积分数据：二维FBP思想怎么推广到三维线积分
- 5.2 平行的面积分数据：为什么会出现三维Radon数据
- 5.3 锥形束数据：为什么锥形束更贴近实际，但几何和精确反演更难
- 5.3.1 Feldkamp算法：先知道它是常用近似锥形束FBP
- 5.3.2 Grangeat算法、5.3.3 Katsevich算法：先知道它们是在处理更严格/更精确的锥形束反演问题

### 第一遍可以先战略性跳过
- 5.4.6 Tuy关系
- 5.4.7 Grangeat关系
- 5.4.8 Katsevich算法

原因：
如果第2章中心切片定理和第3章几何变量都还不稳，直接深抠这几节性价比不高。

### 第二遍再回头补
- 5.4.1, 5.4.2 先反投影后滤波/先滤波后反投影的三维版本
- 5.4.3 三维Radon反演公式
- 5.4.5 Feldkamp算法

### 这一章看图技巧
每看到一张三维几何图，先不要急着看公式，先手动画出：
- 射线是一条线，还是某个平面积分
- 探测器是线探测器、面探测器，还是等价几何里的某个曲面
- 数据是随哪个角度/哪个探测器坐标变化

### 过关标准
你能用自己的话说明：
**为什么从2D到3D以后，不只是“多一个z坐标”这么简单，而是数据完备性、扫描轨迹、探测器几何都会影响是否能精确重建。**

## 第6章 迭代重建

### 这章定位

这是从“解析公式直接反演”转到“把重建写成优化/统计估计问题”的关键一章。对后续看现代深度重建、稀疏重建也很重要。

### 优先吃透
- 6.1 解线性方程组：先建立离散模型 g = Af 的概念
- 6.2 ART算法：把每条射线约束看成逐步修正图像
- 6.3 梯度下降算法：理解“沿着目标函数下降方向更新”
- 6.4 MLEM算法、6.5 OSEM算法：理解为什么发射型成像里常见
- 6.6 噪声控制：迭代次数、像素模型、系统建模都会影响噪声/分辨率
- 6.8 利用先验知识：这是正则化/MAP思想的入口
- 6.10 利用 l0 极小化重建极度欠采样数据：第一遍先理解“稀疏先验 + 欠采样重建”这个思想，不必先追所有细节

### 第二遍重点补的推导
- 6.9.1 ART
- 6.9.3 MLEM
- 6.9.4 OSEM
- 6.9.5 MAP

### 第一遍可以先不深抠
- 6.9.2 共轭梯度算法
- 6.9.6 匹配的和不匹配的投影运算与反投影运算对

### 这一章最重要的思维转变
解析重建更像“直接写出反演公式”，迭代重建更像“先写正向模型和目标函数，再一步步求最优图像”。

也就是说，你要反复把每个算法翻译成这个模板：

**已知数据 g，系统矩阵/正向算子 A，未知图像 f，目标函数 J(f)，更新公式 f^{k+1} = update(f^k, A, g)。**

### 过关标准
你能解释：
**FBP 和 ART/MLEM 最大的思想差别是什么，为什么迭代法更容易引入先验和物理建模，但计算代价通常更高。**

## 第7章 MRI中的图像重建

### 这章定位

这一章的关键不是X射线几何，而是弄清楚MRI为什么天然和k空间/傅里叶编码联系在一起。

### 优先吃透
- 7.1 磁 “M”：先知道磁化矢量和静磁场的大概物理图像
- 7.2 共振 “R”：为什么会有拉莫尔频率/射频激发这个概念
- 7.3 成像 “I”：最重要
  - 7.3.1 断层选择
  - 7.3.2 频率编码
  - 7.3.3 相位编码

### 学这一章的核心问题
你要反复问：

**MRI是怎么把空间位置编码进信号频率/相位里，最后为什么能通过k空间采样和傅里叶逆变换回到图像？**

### 第一遍对 7.4 数学表达式 的建议
先抓“空间编码 -> k空间轨迹 -> 逆傅里叶重建”这条线，不必先把每个物理常数都记死。

### 过关标准
你能讲清楚：
**CT里常讲投影和Radon变换，MRI里为什么更常讲k空间和傅里叶反变换。**

## 四、你最需要优先补的数学基础

建议边读边补，不要先脱离书单独补半年数学。

### 1. 傅里叶分析
必须掌握：
- 一维/二维傅里叶变换的基本意义
- 卷积定理
- 低频/高频和图像平滑/边缘的关系
- 采样不足为什么会带来伪影或混叠

对应书中重点：
- 第2章
- 第7章

### 2. 线性代数和反问题
必须掌握：
- 矩阵乘法表示线性正向模型
- 超定/欠定问题
- 最小二乘思想
- 病态问题为什么对噪声敏感

对应书中重点：
- 第6章

### 3. 概率统计和优化
必须掌握：
- 噪声模型
- 似然函数、先验、MAP的直觉
- 梯度下降更新

对应书中重点：
- 第6章
- 第4章中的发射型成像

### 4. 基本成像几何
必须掌握：
- 平行束、扇形束、锥形束分别是什么
- 射线角度、探测器坐标、源-探测器几何怎么画

对应书中重点：
- 第1、2、3、5章

## 五、看不懂公式时的处理流程

遇到某个推导卡住，不要直接判定“我数学不行”，按下面流程拆：

1. 先写一句话：这个公式在求什么？
2. 把所有符号列成表：每个变量属于图像域、投影域、频域，还是迭代索引？
3. 标出从上一行到下一行用了什么操作：
   - 傅里叶变换/逆变换
   - 卷积定理
   - 坐标变换
   - δ 函数筛选积分路径
   - 求导/梯度
   - 极大似然或贝叶斯推断
4. 如果还是断了，就自己构造一个“最简单例子”：
   - 先用一维信号替代二维图像
   - 先用2×2或3×3像素的小矩阵替代大图
   - 先只看一个角度、一条射线、一次迭代更新
5. 把推导补出来以后，立刻写一行“工程翻译”：
   - 这一步在程序里对应滤波、反投影、插值、矩阵乘法，还是一次参数更新？

### 一个特别重要的提醒

很多“图看不懂”其实不是视觉问题，而是你还没把**符号、几何对象、数据域**三者对上。

## 七、建议的8周执行版计划

如果你想尽快推进，可以先按这个8周版本走第一轮+半个第二轮。

### 第1周
- 第1章完整读完
- 第2章先读 2.1-2.3
- 输出一页笔记：投影、反投影、中心切片定理分别在说什么

### 第2周
- 第2章读完 2.4-2.7
- 重点补 2.6.3 和 2.6.4
- 专门整理一页“FBP为什么要滤波”

### 第3周
- 第3章完整读一遍
- 第4章读 4.1-4.3
- 自己画一张“平行束 vs 扇形束 vs CT/PET/SPECT数据来源”对比图

### 第4周
- 第4章读完
- 第6章读 6.1-6.5
- 建立 g = Af 和 J(f) 两个基本模板

### 第5周
- 第6章读 6.6-6.10
- 第二遍补 6.9.1、6.9.3、6.9.5
- 写一页对比：FBP、ART、MLEM、MAP各自适合什么情况

### 第6周
- 第7章完整读一遍
- 重点整理“断层选择、频率编码、相位编码、k空间、逆傅里叶重建”

### 第7周
- 回到第5章，先读 5.1-5.3
- 对 5.4 只挑 Feldkamp 相关内容先建立印象
- 目标不是一次吃透 Katsevich，而是先能画出3D扫描几何

### 第8周
- 回头复盘第2、3、6章
- 每章至少挑2道例题或练习题认真做
- 把所有仍看不懂的图和公式列成一个“问题清单”，逐个攻破

## 八、每次学习时的固定动作

建议每次学习都按这个流程来，效率会比散读高很多：

1. 先用10分钟复述上一节讲了什么
2. 再用40-60分钟读新内容，只抓主线和关键公式
3. 用20分钟专门处理一张图或一个推导
4. 最后用10分钟写“今天真正新增理解了什么，还有哪里不懂”

最有效的做法是：
